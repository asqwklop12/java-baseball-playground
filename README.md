## 숫자 야구 게임 피드백

기본적으로 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임이다.

- [ ] 플레이어로부터 받은 3개의 숫자 예외 처리
    - [ ] 1 ~ 9의 숫자인가?
    - [ ] 중복 값이 있는가?
    - [ ] 3자리인가?
- [] 컴퓨터는 랜덤으로 3개의 숫자를 생성한다.
- [] 컴퓨터의 숫자와 플레이어의 숫자를 비교한다.
    - [ ] 숫자 한개씩 비교 했을 때
      - [ ] 위치와 숫자를 입력 받는다.
    - [ ] 컴퓨터는 3자리 숫자를 입력하고 플레이어는 숫자 하나씩 비교한다.
      - [ ] 숫자만 같으면 점수+1 
      - [ ] 숫자 위치 모두 같으면 점수+10
      - [ ] 모두 틀릴경우 notting
    - [] 위치가 같으면 스트라이크, 위치가 다르면 볼을 반환한다.
      - [] 점수가 1점이면 볼
      - [] 점수가 2점이면 2볼
      - [] 점수가 3점이면 3볼
      - [] 점수가 10점이면 1 스트라이크
      - [] 점수가 20점이면 2 스트라이크
      - [] 점수가 30점이면 3 스트라이크
- [] 3개의 숫자를 맞추면 게임을 종료한다.
